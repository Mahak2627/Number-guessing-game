# Number-guessing-game
A number Guessing Game 
<br>
Author: Mahak Sharma
<p>
Welcome to the Number Guessing Game!
In this game, the program randomly selects a number and challenges the player to guess it correctly.
<ul>
<li>Guess too low? You’ll be asked to <b>enter a higher number</b>.</li>
<li>Guess too high? You’ll be asked to <b>enter a lower number</b>.</li>
</ul>
The game continues until the correct number is guessed, after which the program reveals <b>how many attempts it took to win</b>.
</p>
